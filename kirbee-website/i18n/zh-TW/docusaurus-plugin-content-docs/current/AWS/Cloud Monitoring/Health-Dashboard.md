# 健康仪表盘 - 服务历史

- 显示所有区域，所有服务健康
- 显示每天的历史信息
- 您可以订阅的 RSS 源
- 之前称为AWS 服务健康控制面板

# 健康仪表盘 - 您的帐户

- 之前称为AWS 个人健康仪表板 (PHD)
- 当AWS遇到可能影响您的事件时，AWS账户健康控制面板提供警报和补救指导。
- 当服务健康仪表板显示AWS服务的一般状况时， 帐户健康控制面板为您提供了一个个性化的视图，显示您所在的 AWS 服务的性能和可用性
- 仪表盘显示相关和及时的信息来帮助您预定的 astite，提供主动的通知来帮助
- 可以从整个AWS组织收集数据
