# 渗透测试

- 欢迎AWS客户在未经事先批准的8项服务之前，对其AWS基础设施进行安全评估或渗透测试：
  - Amazon EC2 实例, NAT 网关, 和 Elastic 负载平衡
  - Amazon RDS
  - 亚马逊云台
  - Amazon Aurora
  - Amazon API 网关
  - AWS Lambda 和 Lambda 边缘函数
  - Amazon Lightsail资源
  - Amazon Elastic Beanstalk环境

## 禁止的活动

- 通过 Amazon Route 53 托管区步行的 DNS 区域
- 拒绝服务（DoS），分配拒绝服务（DS），模拟DoS，模拟DDoS
- 3. 港口水灾问题
- F. 礼宾水灾问题
- 请求水灾(登录请求洪水，APL请求洪水)
