# 一. 导言

- 在 AWS 上启动第三方高性能文件系统
- 完全管理服务

## Windows 文件服务器的Amazon FSx

- 一个完全管理、高度可靠和可扩展的 Windows 本机共享文件系统
- 基于 Windows 文件服务器
- 支持SMB 协议和Windows NTFS
- 与 Microsoft Active Directory 集成
- 可以从 AWS 或您的前提基础结构访问

## Lustry的Amazon FSx

- 高性能计算（HPC）完全管理的高性能、可缩放的文件存储
- Lustre 名称来自“Linux”和“cluster”
- 机器学习、分析、视频处理、金融模型...
- 缩放最多100sGB/秒，数百万IOPS，子ms 延迟。
