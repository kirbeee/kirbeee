# AWS 批量

- 在任何规模上管理完备的批量处理
- 在 AWS 上高效率地运行 100 000 个计算批处理任务
- “批量”作业是一个开始和结束的作业(相对于连续)
- 批处理将动态启动 EC2 实例或Spot 实例
- AWS 批量提供计算/内存的正确金额
- 您提交或计划分批作业，AWS 批处理其余部分！
- 批量作业定义为 Docker 图像并在 ECS 上运行
- 有助于优化成本，较少注重基础设施

# AWS 批处理与 Lambda

## AWS 批量

- 无时间限制
- 任何运行时间，只要它作为停靠的图像
- 在 EBS / 实例存储磁盘空间
- 依赖于EC2(可以由 AWS管理)

## 兰巴达

- 15 分钟限制
- 限时跑步
- 有限的临时磁盘空间
- 无服务器
